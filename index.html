Projects



The Tufts Daily App
The Tufts Daily app allows users to browse articles posted to the Tufts Daily website using a modern and intuitive interface. Users can also browse menus for Tufts Dining halls and track the location of the various Tufts shuttles.
The app was built using React Native and was part of JumboCode 2017






GeoTufts
GeoTufts is a Tufts-based location guessing game. Players are shown pictures of 10 locations on campus and place a marker on a map where they think the picture was taken. The closer they are, the more points they get If they get stuck they can use hints, which cost points to use.
The app was built using EJS, Node.JS, and MongoDB

GeoTufts was my final project for my Web Programming class







Homework X

Homework X is a homework tracker tool that I use to stay on top of my assignments
As due dates approach assignments change color from green to yellow to orange to red. Their corresponding emojis also become more intense.
Assignments can be sorted by due date or class and notification reminders can be set.
I built this app as a way to keep track of homework and dive into mobile development. The app was made using Swift
